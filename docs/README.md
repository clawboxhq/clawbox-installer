## GitHub Pages

The `pages/` directory contains files for deploying ClawBox to GitHub Pages.

### Files

| File | Purpose |
|------|---------|
| `index.html` | Landing page |
| `CNAME` | Custom domain configuration (optional) |
| `docs.html` | Documentation redirect |
| `robots.txt` | SEO configuration |
